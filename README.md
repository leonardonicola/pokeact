## 🏃💨 Getting started

1. Clone this repo with ``git clone https://github.com/leonardonicola/pokeact.git``
2. Move yourself to the pokeact directory: `cd pokeact`
3. Run the ``npm i`` to add all dependencies
4. Run ``npm start`` to start the app
